# 1 - Cenário

```
Uma papelaria deseja fazer um catalogo online, onde clientes podem ver os produtos ofertados pela loja e possiveis descontos de produtos

```

### OBS: Os Donos da loja tem um conhecimento parcial em tecnologias, mas desejam algo bonito simples e facil de usar

# 2 Brifeing

* **Qual o objetivo do site?**
```
Criar um catalogo online simples e intuitivo para divulgar produtos e promoções
```


* **Quem é o publico alvo?**
```
 pais e jovens adultos que buscam materiais escolares, de escritorio 
```

* **O que o site precisa ter?**

```
Painel simples para a atualizaçao de produtos
Listagem de produtos com imagens preço e descriçao
Destaque de promoçoes
Botão de contato via Whatsaap
Categorias de produtos (Cadernos,Mochilas,Lapis...)
Sistema de login (para administradores)
```

# Requisitos

### Funcionalidade

```
* O sistema deve permitir o cadastro de novos produtos
* O sistema deve permitir editar e excluir produtos
* O site deve permitir o cadastro de novos produtos
* O site deve der um botão que redirecione o cliente para o WhatsApp
```

### Usabilidade

```
* A interface do site deve ser simples e intuitivo
* As promoções devem aparecer em destaque ao cliente entrar na pagina
* O usuário deve encontrar produtos com poucos cliques
* O botão de contato via WhatsApp deve estar sempre visível
```

### Confiabilidade

```
* O sistema deve estar disponível na maior parte do tempo (alta disponibilidade)
* O site deve seguir as normas de LGPD
* Deve haver backup periódico das informações
* Os dados devem ser armazenados com segurança
```

### Desempenho    

```
* O site deve ser fluido e rapido
* O sistema deve ser leve e não consumir muitos recursos do dispositivo
* As imagens devem ser otimizadas para carregamento rápido
```

### Suportabilidade

```
* O site deve ser compativel com diferentes dispositivos(tablets, celulares e diferentes tamanhos de tela)
* O sistema deve oferecer recursos de acessibilidade (alto contraste, ajuste de fonte, HTML semântico)
* O código deve ser organizado para facilitar manutenção
```

### +?

```
* O sistema será desenvolvido como uma aplicação web
* O painel admnistrativo deve ser acessivel apenas para usuarios autorizados
* A integração com o WhatsApp sera feita via link direto

```
